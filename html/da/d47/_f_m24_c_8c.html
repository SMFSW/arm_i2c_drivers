Detailed Description

FRAM / EEPROM Driver (bank switching at I2C address level protocol)

Author
SMFSW
Copyright
MIT (c) 2017-2023, SMFSW
Note
The driver handles writing specificities for EEPROM type components
The driver is fully compatible with FRAM type components
When EEPROM / FRAM compatibility is not needed, FM24C_WRITE_SIZE can be set to FM24C_BANK_SIZE for more efficiency

FM24C16B: 16-Kbit (2K * 8) Serial I2C F-RAM FM24C04B: 4-Kbit (512 * 8) Serial I2C F-RAM ...

Note
Compatibility (tested):
  • FM24C16B
  • FM24C04B
  • BR24T04FVM
