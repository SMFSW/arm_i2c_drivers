FctERR ADS1115_handler ( ADS1115_t *  pCpnt)

Handler for ADS1115 peripheral.

Weak Functions:
ADS1115 handler may be user implemented to suit custom needs
Note
May be called periodically to handle ADS1115 tasks
Alternately may be called when event occurs on ADS1115 pin
Parameters
[in]pCpnt- Pointer to ADS1115 component
Returns
FctERR - error code
Note
In case of ADS1115__MODE_CONTINUOUS, conversion time need to be respected (thus ADS1115_get_conv_ms may be used to help ensure that)
