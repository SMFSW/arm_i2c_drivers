Detailed Description

FM24C Driver extensions.

Author
SMFSW
Copyright
MIT (c) 2017-2023, SMFSW

FM24C16B: 16-Kbit (2K * 8) Serial I2C F-RAM FM24C04B: 4-Kbit (512 * 8) Serial I2C F-RAM

Note
Compatibility (tested):
  • FM24C16B
  • FM24C04B
  • BR24T04FVM
Compatibility:
  • other components using same i2c protocol may be compatible
