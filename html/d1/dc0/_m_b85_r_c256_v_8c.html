Detailed Description

MB85RC256V Driver.

Author
SMFSW
Copyright
MIT (c) 2017-2020, SMFSW

MB85RC256V: 256-Kbit (32K * 8) I2C Memory FRAM

Note
Compatibility (tested):
  • MB85RC256V
  • FM24CL64B
Compatibility:
  • other components using same i2c protocol may be compatible
