Detailed Description

MB85RC256V FRAM Driver.

Author
SMFSW
Copyright
MIT (c) 2017-2024, SMFSW

MB85RC256V: 256-Kbit (32K * 8) I2C Memory FRAM

Note
Legacy component: consider switching to I2CMEM component for new designs
Compatibility (tested):
  • MB85RC256V
  • FM24CL64B
