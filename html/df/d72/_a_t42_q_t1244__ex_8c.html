Calculate Host CRC for AT42QT1244 peripheral.

Note
CRC is calculated using setup block values from component
As the whole setup CRC procedure may take some time, iwdg is automatically refresh if used

After CRC is written, AT42QT1244_Reset is called, leading to a blocking delay described in AT42QT1244_Delay_PowerOn
